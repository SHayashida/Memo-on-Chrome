# Memo on Chrome

![Version](https://img.shields.io/badge/version-1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

**特定のウェブページに付箋メモを貼り付けて、いつでも大切な情報を確認できるChrome拡張機能**

## 📌 解決する課題

ウェブブラウジングをしていて、こんな経験はありませんか？

- 「このページで後で確認したいポイントがある」
- 「このサイトの使い方の注意点をメモしておきたい」
- 「重要な情報を見逃さないように、視覚的にリマインダーが欲しい」
- 「複数のページにメモを残して、作業効率を上げたい」

**Memo on Chrome**は、ウェブページ上に直接メモを貼り付けることができる拡張機能です。紙の付箋のように、気になる情報をその場所に残しておけるので、次回そのページを開いた時にすぐに確認できます。

## ✨ 特徴

### 🎯 ページ単位でメモを管理
各ウェブページごとにメモが保存されるため、サイト固有の情報を効率的に管理できます。

### 🔄 折りたたみ可能なUI
- メモは**アイコン（!）**として折りたたんで表示可能
- ページの閲覧を邪魔せず、必要な時だけ展開して確認できます
- 折りたたみ状態も保存されます

### 🖱️ ドラッグ＆ドロップ
メモを自由に移動できるため、最適な位置に配置できます。位置情報も自動保存されます。

### 💾 自動同期
Chrome の同期機能により、複数のデバイス間でメモを共有可能（Chrome Sync 有効時）

### 🎨 シンプルで直感的
複雑な設定は不要。すぐに使い始められるシンプルなインターフェース。

## 📸 使用例

### 使い方のイメージ
1. ウェブページを開く
2. 拡張機能アイコンをクリックしてメモを入力
3. ページ上に「!」アイコンが表示される
4. アイコンをクリックして展開・確認
5. ドラッグして好きな位置に移動

## 🚀 インストール方法

### 開発版のインストール（現在）

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/yourusername/Memo-on-Chrome.git
   ```

2. Chrome ブラウザで `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」を有効にする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

6. インストール完了！🎉

## 💡 使い方

### メモの追加
1. メモを追加したいページで、拡張機能アイコン（ツールバー）をクリック
2. ポップアップが開くので、メモの内容を入力
3. 「メモを追加」ボタンをクリック
4. ページ上に「!」アイコンが表示されます

### メモの表示・折りたたみ
- **「!」アイコンをクリック** → メモが展開表示されます
- **「−」ボタンをクリック** → メモが折りたたまれ、アイコン状態に戻ります

### メモの移動
- メモ（アイコンまたはコンテンツ部分）をドラッグして、好きな位置に移動できます
- 位置は自動的に保存されます

### メモの削除
- 展開したメモの「×」ボタンをクリックすると削除されます

## 🛠️ 技術スタック

- **Manifest V3** - 最新の Chrome 拡張機能仕様
- **Chrome Storage Sync API** - デバイス間でのデータ同期
- **Content Scripts** - ページへの動的なメモ挿入
- **Vanilla JavaScript** - 軽量でシンプルな実装

## 📂 ファイル構成

```
Memo-on-Chrome/
├── manifest.json          # 拡張機能の設定ファイル
├── popup.html             # メモ入力用ポップアップ画面
├── popup.js               # ポップアップの動作制御
├── content_script.js      # ページ上でのメモ表示・管理
├── memo.css               # メモのスタイル定義
└── icons/                 # 拡張機能のアイコン
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 🔒 プライバシー

- メモは **Chrome のストレージ** に保存され、外部サーバーには送信されません
- ページの URL とメモの内容のみを保存します
- 個人情報の収集は一切行いません
